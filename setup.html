üìä Google Sheets Integration (Recommended)

For production use with real-time data sync across multiple users:

Step 1: Create Google Spreadsheet

  1. Go to Google Sheets
  2. Create a new spreadsheet
  3. Copy the Spreadsheet ID from the URL (between /d/ and /edit)

Step 2: Enable Google Sheets API

  1. Go to Google Cloud Console
  2. Create a new project or select existing one
  3. Enable the Google Sheets API
  4. Create credentials (API Key)
  5. Copy the API Key

Step 3: Configure the Application

  1. Open google-sheets.js
  2. Replace YOUR_SPREADSHEET_ID with your actual Spreadsheet ID
  3. Replace YOUR_API_KEY with your actual API Key
// In google-sheets.js, update these values: this.SPREADSHEET_ID = 'your-actual-spreadsheet-id'; this.API_KEY = 'your-actual-api-key';
